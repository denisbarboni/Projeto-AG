    Overview

    Base class for ranking and creating populations

    Property Summary

    All the genomes of this population

    (Float) mutationChance

    The chance of a Genome to mutate

    (Boolean) elitism

    If true, the two best Genomes will survive without mutation or mating

    (Float) mixingRatio

    Determines the amount of values to be copied from parents

    (Integer) currentGeneration

    The number of participants for the tournament selection
